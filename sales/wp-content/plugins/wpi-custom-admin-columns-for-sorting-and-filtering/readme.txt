This plugin adds the capability to add custom columns to any post type to provide sorting and filtering of posts using those columns.
== Description ==

**Features**

1. Ability to add custom columns per post type,
2. Ability to sort posts in the admin panel through the custom columns,
3. Ability to filter posts in the admin panel through the custom columns,

For detailed instructions on using this plugin visit: http://phpwpinfo.com/wordpress-plugin-admin-custom-columns-for-sorting-and-filtering



== Installation ==

1. Upload the plugin files in wp-content/plugins folder (same as any other wordpress plugin)
2. Activate the plugin
